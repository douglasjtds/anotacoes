Localização:
	Interna ou externa.
Capacidade:
	Expresso em bytes ou palavras
	Tamanhos comuns de palavras: 8, 16 e 32 bits
Unidade de Transferência:
	Pode ser o tamanho da palavra
	Normalmente é maior: 64, 128 ou 256 bytes

Método de Acesso:
	
	- Acesso sequêncial: acesso em uma sequência linear
		Dados organizados em registros sequenciais
		Exemplo: fitas magnéticas.
		Tempo de acesso variável
	
	- Acesso direto: cada bloco de dados possui um endereço único
		Baseado na localização física
		Acesso feito através de um acesso direto a uma vizinhança genérica do registro, e em seguida busca sequencial
		Tempo de acesso variável.  Ex.: HD

	- Acesso aleatório: cada posição de memória possui um endereço único
		Tempo de acesso a uma posição é constante.
		Ex.: memória principal (RAM) e alguns sistemas de memória cache.

	- Associativo: acesso aleatório que compara certo número de bits de uma palavra com todas as palavras em memória.
		Determina quais posições possuem o mesmo padrão de bits
		Uma palavra é buscada com base em parte de seu conteúdo, não de acordo com o seu endereço.
		Ex.: memórias cache.


Desempenho:
	- Tempo de Acesso (latência):
		- Em memórias de acesso aleatório:
			Tempo decorrido desde o instante em que um endereço é apresentado à memória até o momento em que os dados são armazenados ou se tornam disponíveis para utilização.
		- Em memórias de acesso não-aleatório.
			Tempo gasto para posicionar o mecanismo de leitura-escrita na posição desejada.

	- Tempo de ciclo de memória:
		- Aplicável principalmente às memórias de acesso aleatório
		- Compreende o tempo de acesso + um tempo adicional antes de um segundo acesso
		- Necessário para o desaparecimento de transientes (ruídos) nas linhas de sinal
		- Esse tempo refere-se ao barramento e não ao processador

	- Taxa de transferência:
		- Taxa no qual os dados podem ser transferidos de ou para a unidade de memória.
		- Para memórias de acesso aleatório: 1/tempo_de_ciclo
		- Para memórias de acesso não-aleatório: 
			Tn = Ta + (n/R)
				Tn = Tempo médio para ler ou escrever N bits
				Ta = Tempo de acesso médio
				n = número de bits
				R = taxa de transferência em bits por segundo (bps)


Tecnologia: 
	- Semicondutora (mais comuns)
		RAM
	- Memória de superfície magnética
		Disco ou fita
	- Óptica
		CD/DVD
	- Magneto-óptica
		Holograma

Características Físicas: 
	- Volátil: informação se deteriora naturalmente ou se perde quando a energia elétrica é desligada (pulso).
	- Não-volátil: informação gravada permanente sem deteriorização.
	- Não-apagável: memória somente leitura (ROM)
